# WRH Pilot Package

## What Really Happened™

**WRH is structured non-clinical survival-mode education with safety boundaries and measurable pilot delivery.**

This pilot package gives administrators, program directors, reentry leaders, veteran-serving organizations, recovery-support programs, and community partners a bounded way to review WRH before broader implementation.
