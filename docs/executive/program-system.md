# Program System

## Dark Mode Executive Edition

The WRH Program System is the layer that turns the WRH model into teachable, repeatable, agency-ready delivery.

If the Survival Map explains where the pattern came from, and the Trajectory Engine explains how direction changes, the Program System explains:

> How do we teach this?  
> How do we deliver it?  
> How do we make it repeatable?

The Program System includes four components:

- Staff Training
- Participant Workshops
- Implementation Support
- Evaluation System

Each component serves a different audience and a different function inside an agency.

---

## 1. Staff Training

Staff Training is the foundation of WRH program delivery.

It gives staff:

- a shared language
- a mechanical model
- a way to understand participant behavior without blame
- a coaching framework
- a method for identifying forks and interrupts
- a structure for supporting directional change

Staff Training includes:

- The Survival Map
- The Trajectory Engine
- Applied WRH
- Leadership & Fidelity

This ensures the entire team is aligned on how WRH works and how to use it.

---

## 2. Participant Workshops

Participant Workshops translate the WRH model into direct service.

Workshops focus on:

- pattern visibility
- fork recognition
- interrupt practice
- safer replacements
- evidence building
- directional stability

Workshops are:

- non-clinical
- non-exposure-based
- non-confessional
- non-punitive

They give participants a mechanical, not emotional, way to understand their patterns and build direction.

---

## 3. Implementation Support

Implementation Support ensures WRH is delivered consistently across staff, shifts, and programs.

It includes:

- coaching
- fidelity monitoring
- case consultation
- staff support
- integration with existing workflows
- alignment with agency goals

Implementation Support is what makes WRH stick inside a system.

Without it, WRH becomes a training.

With it, WRH becomes a program.

---

## 4. Evaluation System

The Evaluation System measures:

- interrupt frequency
- evidence accumulation
- directional stability
- crisis utilization
- retention
- relapse/recidivism
- participant movement

This gives agencies:

- early indicators
- mid-range indicators
- long-range outcomes

The Evaluation System turns WRH from a good idea into a measurable program.

---

## Why the Program System Matters

The Program System is the bridge between:

- theory: Survival Map and Trajectory Engine
- delivery: training and workshops
- implementation: coaching and fidelity
- measurement: evaluation and reporting

It ensures WRH is:

- teachable
- repeatable
- scalable
- measurable
- fundable
- sustainable

This is the layer that agencies interact with most directly.

---

## Summary Line

The WRH Program System turns the WRH model into a repeatable, measurable, agency-ready program that staff can deliver and participants can use.
