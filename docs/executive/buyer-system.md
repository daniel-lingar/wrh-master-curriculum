# WRH Buyer-Facing System

The WRH Buyer-Facing System organizes the materials used for directors, agencies, funders, and implementation partners.

Core components:

- One-Page Sell Sheet
- Product Catalog
- Training Catalog
- Pilot Program Sheet
- Pricing Sheet
- Statement of Work
- License Terms
- Evaluation Framework
- Reporting Tools
- Session 16 Sample Packet
