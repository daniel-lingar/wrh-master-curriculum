# WRH Federal System

The WRH Federal System organizes grant, funder, and agency-facing materials.

Core components:

- Federal Abstract
- Needs Statement
- Project Description
- Goals & Objectives
- Logic Model
- Work Plan / Timeline
- Budget Justification Narrative
- Performance Measurement Plan
- Evaluation Plan
- Organizational Capacity Statement
- Sustainability Plan
- SAMHSA Compliance Language
- Letters of Support Templates
