# Legacy 12-Session Rhythm

Source: harvested and cleaned from the legacy `What-Really-Happened-Manual` short-form facilitator version.

Purpose: preserve the useful short-form session rhythm so it can inform future introductions, workshops, condensed pilots, or facilitator training.

## Use Boundary

This rhythm is a legacy short-form structure. It does not replace the 76-session master curriculum.

It may be useful for:

- short workshops
- introductory sessions
- pilot previews
- facilitator training
- simplified public education formats

## Standard Session Flow

| Segment | Purpose |
|---|---|
| Cold open | names the theme and creates attention |
| Quick grounding | orients the room before content deepens |
| Core concept | teaches the mechanism or pattern |
| Mirror moment | invites private recognition without forced disclosure |
| Rapid reframes | replaces shame-based labels with mechanism-based language |
| Micro-regulation pause | lets the room settle before closing |
| Retention anchor | gives the participant one phrase or concept to carry |
| Open loop | invites observation between sessions |
| Cliffhanger / bridge | connects the current session to the next step |

## Core Design Principle

The session should move from recognition to language to regulation.

It should not move from recognition to forced disclosure.

## Mirror Moment Rule

Mirror questions should invite private recognition.

They should not pressure participants to share details publicly.

Example frame:

> Notice if this pattern is familiar. You do not have to explain why.

## Rapid Reframe Rule

Rapid reframes should be short, clear, and non-shaming.

They should translate a label into a mechanism.

Example pattern:

```text
Label -> mechanism-based explanation
```

## Open Loop Rule

The open loop should ask participants to observe one behavior or pattern before the next session.

It should not demand repair, confession, or performance.

Example frame:

> Between now and next time, notice when this pattern shows up. Do not fix it yet. Just notice what came right before it.

## Why This Matters

The legacy rhythm contains one of the strongest WRH design instincts:

> The participant does not need to expose the wound to learn the map.

That principle should remain active across the full curriculum.
