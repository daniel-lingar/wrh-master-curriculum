## Standard Session Structure

Each new WRH session should include:

1. Facilitator Notes
2. Session Purpose
3. Participant Takeaway
4. Scope Boundary
5. Opening Script
6. Grounding / Regulation Moment
7. Core Teaching Section One
8. Core Teaching Section Two
9. Lived-Pattern Anchor from *From the Storm to the Fire*
10. Mechanism Map
11. Reflection Prompt
12. Practical Tool / Reframe
13. Integration Script
14. Closing Script
15. Cliffhanger into the next session
16. Implementation Notes

## Rule

The new system is not a topic guide. It is a word-for-word facilitator script system.
