# WRH New System — Word-for-Word Facilitator Scripts

This folder holds the rewritten WRH curriculum in the new standardized delivery format.

## Purpose

The old WRH structure is the curriculum map: session titles, architecture, and topic sequence.

The new WRH structure is the delivery system: complete word-for-word facilitator scripts that can be read consistently in any location.

## Core Standard

A finished WRH session must be:

- non-clinical
- no-disclosure
- word-for-word readable
- mechanism-first
- plain-language
- grounded in lived pattern examples when useful
- shame-reducing without excuse-making
- accountability-preserving
- structured with a regulation moment, mechanism map, reflection, practical tool, closing script, and cliffhanger
